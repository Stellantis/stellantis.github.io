pims.authentication.user - forcedLogin

This api allows to force the login. It will ALWAYS prompt an external user agent (web view) even if the session is still open on the server.

Input Parameters Object
Param name Description Type Unit/Values Example Required
action

Name of the action to perform, in this case forcedLogin.

String

forcedLogin

 		forcedLogin true
local

Define the language to use in the external user agent (web view).

String

 en false
context

Context. Only for Android.

Object

n/a

 		context true
Output Data Object
No outputted data

No errors
